Mac上IDEA的常用快捷键

By Treesouth

发表于 2015-03-21

工欲善其事，必先利其器。

  • 熟记并能熟练使用快捷键，能大大提升编码效率，提高编码快感。

  • 默认快捷键是Mac OS X 10.5+ , 前面加“ * ”的是自定义快捷键，可以在Setting/Keymap中搜索英文描述而直接定位。

  • 诸如Cmd + C, Cmd + V等不再赘述，整理于官方文档。

Editing:

Keymap Description-CN Description-EN
Ctrl + Shift + Space 代码智能完成 Smart code completion
Ctrl + J 代码快速查阅注释 Quick documentation lookup
Cmd + N 生成代码 Generate code…
Ctrl + O 重写代码 Override methods
Alt + Cmd + T 添加“包围”代码块 Surround  with…
Cmd + / 单行注释 Comment/uncomment with line comment
* Ctrl + Cmd + / 多行注释 Comment/uncomment with block comment
Alt + ↑ “蔓延”选择代码块 Select successively increasing code blocks
Alt + Cmd + L 格式化代码 Reformat code
Cmd + X 删除行（其实是剪切） Cut current line or selected block to clipboard
Ctrl + Alt + O 优化导入 Optimize imports
Cmd + Shift + V 从剪贴板选择粘贴 Paste from recent buffers…
Cmd + Del 删除选择代码块所在行 Delete line at caret
Cmd + Shift + V 从剪贴板选择粘贴 Paste from recent buffers…
Cmd + D 复制行 Duplicate current line or selected block
Cmd + “+ / -“ 展开/折叠代码块 Expand/collapse code block
Alt + Del 删除选中单词的前半部分 Delete to word start
Fn + Alt + Del 删除选中单词的后半部分 Delete to word end
Cmd + Shift + U 转换大小写 Toggle case for word at caret or selected block
* Shift + P 重命名 Rename
Shift + Alt + ↑ / ↓ 代码上移/下移 Move line to previous/next
Shift + Alt + “Click” 同时多处编辑
Cmd + Alt + M 将选中代码抽取成新的Method

Search:

Keymap Description-CN Description-EN
Double↑ 搜索一切 Search everywhere
* Ctrl + Q 查找被调用 Find Usage
* Ctrl + Z 显示被调用 Show Usage
Ctrl + G 查找下一处相同变量 Override methods
Ctrl + R 高亮并替换当前变量 Replace
Shift + Cmd + F / R 全文搜索/替换 Find/Replace in path
* Alt + W 查看下一处警告 Next Highlighted error
Alt + P 查看调用的xml文件 Select successively increasing code blocks
* Alt + O 打开Color Picker open Color Picker

Navigation:

Keymap Description-CN Description-EN
Cmd + L 跳转某一行 Go to line
Cmd + O 跳转某个类 Go to class
Cmd + Shift + O 跳转某个文件 Go to file
Alt + Cmd + O 跳转某个变量，方法 Go to symbol
* Alt + L 跳转Finder所在位置 Show in Finder
Cmd + E 显示查阅最近文件 Show Recent Files
Cmd + “{“ / “}” 跳转代码编辑前/后 Jump to last/next edit
Cmd + 0^9 跳转打开相应导航面板 Open corresponding tool window

VCS/Local:

Keymap Description-CN Description-EN
Cmd + V 打开版本控制 “VCS” quick popup
Cmd + K 查看Commit Commit project to VCS

General:

Keymap Description-CN Description-EN
Cmd + , 打开idea设置 Open Settings dialog
Cmd + ; 打开project设置 Open Project Structure dialog
